## 💡 Clipchamp Efficiency Tip

**Build ONE styled template, clone it 17 times:**

1. Create the first label (Scene 1-A) with correct font, size, position, background box, fade-in/out
2. Select it on timeline → right-click → **Save as template** (or just **Duplicate** repeatedly)
3. For each clone:
   - Drag to correct timestamp
   - Double-click text → paste title + subtitle from above
   - Change color if needed (Cyan → Red for Scene 7, etc.)
4. Done in ~30 min instead of 2 hours
